    Smart Typography
    Style Settings
    Text Extractor

And the theme I use is Minimal.

Clip Studio Paint Originally I was using Medibang Paint Pro, but a friend kindly gifted me a copy of CSP (before they changed how it was monetized lol). Firefox My main browser. The only extensions I usually stick with are uBlock, Stylus, Dark Reader, Unhook, and Return Youtube Dislike.

Flow Launcher I don't think I can live without this one.
